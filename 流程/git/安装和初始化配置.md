# 安装和初始化配置
## 安装
1. 前往官网https://git-scm.com 下载git
2. 安装完成后在终端输入git -v
出现版本信息，则安装成功
`PS C:\Users\15779> git -v`
`git version 2.52.0.windows.1`

安装完git后会自动安装git bash，你可以右键文件夹空白处打开它

## 初始化配置
首先打开终端
1. 初始化信息：
`git config --global user.name "这里配置你的名称"`
	-`--global` 全局配置，所有仓库生效
	-`--system` 系统参数，对主机所有用户生效
2. 配置邮箱：
`git config --global user.email "你的邮箱"`

- 保存密码（用于github登录等）
`git config --global credential.helper store`


- 显示配置信息
`git config --global --list`
